# ⚖️ Landing Page - Advogado Previdenciário

Uma landing page moderna e responsiva desenvolvida para apresentação de serviços jurídicos na área de Direito Previdenciário.

## 📸 Preview

Projeto focado em conversão, com destaque para contato rápido via WhatsApp e navegação fluida entre seções.

---

## 🚀 Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**
* **jQuery**
* **ScrollReveal**
* **Font Awesome**

---

## ✨ Funcionalidades

* 📱 **Menu mobile interativo**

  * Abertura e fechamento com botão dinâmico

* 📍 **Scroll inteligente**

  * Destaque automático da seção ativa no menu

* 🎯 **Animações suaves**

  * Elementos surgem conforme o usuário rola a página

* 💬 **Integração com WhatsApp**

  * Botões estratégicos para contato direto

* 🗺️ **Mapa incorporado**

  * Localização do escritório diretamente na página

* 🎨 **Design responsivo**

  * Adaptado para desktop e dispositivos móveis

## 💡 Sobre o projeto

Este projeto foi desenvolvido com foco em **prática de front-end**, utilizando manipulação de DOM com JavaScript e jQuery, além de bibliotecas para melhorar a experiência do usuário.

A landing page tem como objetivo apresentar serviços jurídicos de forma clara, profissional e acessível, incentivando o contato direto com o cliente.
